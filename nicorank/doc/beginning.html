はじめに

ニコニコランキングメーカーはランキング動画作成のための様々な機能を持ったソフトです。まずは以下の導入手順を試してみることをお勧めします。

インストールと起動

ダウンロードしたアーカイブを解凍し、適当な場所に置くだけです。日本語（２バイト文字）を含むフォルダ（デスクトップなど）には置かない方が無難です。

nicorank.exe を起動します（起動には .NET Framework 3.5以降が必要です）。

ニコニコ動画へのログイン

動画のDLやタグ検索などを行うにはニコニコ動画へログインする必要があります。

タグ検索からランク画像生成までの手順

以下の説明は「タグ検索」でランキングを作る場合の手順です。

タブの「リスト作成・更新」をクリック → 「検索するタグ」と書かれたテキストボックスに 好きな検索タグを入力 → 「リスト作成」ボタンをクリック（２回以上押さないように注意して下さい） しばらくすると、リストを作成しましたと表示が出ます。 同じフォルダに rank ファイル（rank.txtファイル）が作成されているので メモ帳かテキストエディタで開いてください。 mylist数の大きい動画順に並べられていることを確認してください。 確認できたらメモ帳は閉じても構いません。

次に FLV 動画のダウンロードを行います。最初は数個だけで試すことをお勧めします。 rank.txt の最初の数行以外を削除してください。

タブの「ファイルDL」をクリック → 「FLV ダウンロード」ボタンをクリック しばらくすると、終了しましたの表示がでます。 この操作では、動画の FLV をダウンロードします。FLV フォルダ内にダウンロードされます。

タブの「FLV 変換」をクリック →「FLV → WAV変換」、「FLVから画像抜き出し」のチェックボックスに チェックを入れる →「変換」ボタンをクリック しばらくすると、終了しましたの表示がでます。 この操作では、FLV → WAV の変換と画像の抜き出しを行います。 made フォルダに作成されます。sm1234567 など ID のフォルダが作成され、 その中に画像が 100 枚ほどできます。

タブの「画像・動画作成」をクリック → 「ランク画像作成」をクリック gen フォルダが作成され、中にランク画像が作成されます。
