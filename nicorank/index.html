ニコニコランキングメーカーとは？

ニコニコランキングメーカーは「ランキング動画」を作成するために必要な機能を提供するプログラムです。 ニコニコ動画の再生数やマイリスト数などのデータの集計から、動画ダウンロード、変換、出力までを支援します。
このソフトはある程度ランキング動画の作成法を知っている人向けです。 特に毎週（毎月）定期的にランキング動画を作る場合に威力を発揮します。
ソフトの情報は少なく、エラーメッセージさえまともに表示されないので、 トラブルが起こった場合に自分で考えて対処できる方でないと使用をお勧めできません。

できること一覧

  • 条件を指定したタグ検索によるランキングリスト作成
  • ニコニコ動画公式ランキングからリスト作成
  • 作成したリストから動画、サムネイルを自動DL、形式変換
  • ポイント等の動画情報を埋め込んだ画像生成
  • マイリストの作成
  • その他いろいろ

必要環境

Windows 7 で動作確認をしています。

.NET Framework 3.5 以上

ダウンロード

sourceforge.jp からダウンロードできます。 初めての方は「ニコニコランキングメーカー本体＋FFmpeg（初めての方はこちら）」をダウンロードしてください。

フレーム画像サンプル　フレーム画像サンプルのページからダウンロードできます。「全部入り」パッケージにも同じものが含まれています。

インストール方法

ZIPファイルを解凍して適当な場所に置いてください。レジストリは一切使いません。アンインストールはプログラム本体を削除するだけです。
